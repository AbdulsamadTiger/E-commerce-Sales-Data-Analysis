This Project analyzes an e-commerce dataset to uncover insights about sales performance, customer behaviour , and product trends. The analysis was peformed using Google Sheets wih several data analysis and functions. 

  TOOLS USED
      Excel/Google Sheets
      Pivot Tables
      Conditional Formatting
      Lookup Functions

   KEY TECHNIQUES APPLIED
       VLOOKUP to retrieve product and category information
       XLOOKUP for advanced data lookup across tables
       QUERY FUNCTION to filter and analyze large datasets
       PIVOT TABLES to summarize sales, revenue,and product performance
       CONDITIONAL FORMATTING to higlight key trends and high-value metrics

   DASHBOARD  
  A dynamic dashboard was created using Pivot Tables to visualize:
Total sales,Top Selling Products, Revenue by Category , Monthly Sales Performance

   PROJECT GOAL
The goal of this project is to demonstrate data analysis skills using spreadsheet tools to transform raw sales data into meaningful business insights.

   DATASET
 Sample e-commerce dataset used for analysis and dashboard creation.
